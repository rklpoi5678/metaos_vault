## 컴포넌트가 좋은 이유
1.반복적인 업무가 줄어든다.
2.고장을 고치기 쉽다.
3.일을 쉽게 나눌수있다. 협업하기 좋다.

>>> 브라우저의 최상단객체는 window객체
>>> 웹 페이지의 최상단객체는 DOM객체
>>> 사실 알고보면 window안에 DOM이들어있기는한데...
