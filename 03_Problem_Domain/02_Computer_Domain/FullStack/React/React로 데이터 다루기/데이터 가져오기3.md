## 조건부 렌더링 꿀팁
**논리 연산자 활용하기**
`AND연산자`
```jsx
import {useState} from 'react';

function App() {
	const [show, setShow] = useState(false);
	
	const handleClick = () => setShow(!show);
	
	return(
		<div>
			<button onClick={handleClick}>토글</button>	
			{show && <p>보인다</p>}
		</div>	
	);
}

export default App;
```
