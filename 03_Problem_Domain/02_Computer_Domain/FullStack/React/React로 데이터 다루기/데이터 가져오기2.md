## useEffect 살펴보기
**처음 한 번만 실행하기**
```jsx
useEffect(() => {
	// execute code
},[])
```
> 첫 렌더싱 되고 나면 리액트가 콜백 함수를 기억해뒀다가 실행
> 그 이후로는 콜백 함수를 실행하지 않는다.

**값이 바뀔 때마다 실행하기**
```jsx
useEffect(() => {
	// execute code
},[dep1,dep2,dep3,...]);
```
> ... 그 이후로 렌더링시 디펜던시 리스트에 있는 값들을 확인해서
> 하나라도 바뀌면 콜백 함수를 기억해뒀다가 실행한다.
