```bash
- 한국 영화

    - 1950년대: 침체기
    - 1960년대: 황금기
    - 1970년대: 쇠퇴기
    - 1980년대: 암흑기
    - 1990년대: 전환점
    - 2000년대: 르네상스
    - 2010년대: 최전성기
    - 2020년대: 대격변
  - 유명한 한국 감독
    - 박찬욱
    - 봉준호
    - 임권택
    - 김기덕
    - 김기영
    - 최동훈
```
