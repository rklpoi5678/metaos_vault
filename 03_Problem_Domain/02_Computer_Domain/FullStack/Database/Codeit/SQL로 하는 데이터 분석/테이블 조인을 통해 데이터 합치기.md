## Foreign Key란?
테이블을 참조할수있게 하는 연결고리 역할을 하는것이 외래키이다.
데이터베이스 구조와 테이블 간의 관계를 파악하기가 훨씬 수월하다.

## 조인할 때 테이블에 alias붙이기
```sql
SELECT 
	i.id,
	i.name,
	s.item_id,
	s.inverntory_count
FROM item AS i RIGHT OUTER JOIN stock AS s
ON i.id = s.item_id
```
