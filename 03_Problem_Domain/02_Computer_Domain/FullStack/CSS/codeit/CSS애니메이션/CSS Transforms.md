## 핵심
주변 요소들의 영향을 미치지 않고 특정 요소의 크기,형태,위치 등을 변경할 수 있다.
transfrom: transform-function;를 적어서 사용할수 있는데, transform-function은 특별한 CSS  함수가 들어가는 부분
