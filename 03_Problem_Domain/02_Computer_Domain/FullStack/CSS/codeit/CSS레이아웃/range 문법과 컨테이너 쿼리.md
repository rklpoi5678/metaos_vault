## 미디어 쿼리 Range 문법
```CSS
@media (min-width: 500px) and (max-width: 600px)
/*500px 이상 600px 이하인 경우에만 미디어쿼리를 적용하겠다.*/
```
다만 이것이 복잡해보여
```CSS
@media (500px <= width <= 600px) {
	..
}
```
이렇게 부등호로 나타낼수있는데 이 문법은 2023년 부터 본격적으로 지원하기 시작해서 , 옛날 버전 웹 브라우저에서는 지원하지 않을 수도 있다.
