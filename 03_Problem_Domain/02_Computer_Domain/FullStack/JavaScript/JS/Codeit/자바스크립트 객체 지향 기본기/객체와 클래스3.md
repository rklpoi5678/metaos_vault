## 부모클래스의 메소드가 필요하다면?
```js
class User ... {
	...
	buy(item) {
		console.log(`${...}${...} asd`)
	}
}`

class PremiumUser extends User {
...
	buy(item){
		super.buy(item); // 부모의 내용과 똑같은 메소득가필요하다. super이용
		this.point += item.price * 0.05;*
	}
}`
```
자식 클래스와 부모클래스를 오버라이딩할수있으며,오버라이딩 한 클래스에서 부모의 클래스의 메소드를 그대로 호출하는것도 가능하다.
