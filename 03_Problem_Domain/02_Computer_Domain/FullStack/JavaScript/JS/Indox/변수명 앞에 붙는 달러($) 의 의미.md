## jquery vs 관습
jQuery에서 일반적으로 구분하기 위해 $를 붙인다는 것이지, $붙은게 jQery를 사용했다는 것은 절대 아니다.

JS에서는 변수 선언 시 달러나 언더바의 경우 변수 앞에 붙이는게 허용된다. 결국은 관습이라는것이다.
